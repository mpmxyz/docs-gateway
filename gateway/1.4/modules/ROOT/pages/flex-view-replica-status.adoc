In Runtime Manager, you can view the statuses of groups of Flex Gateway replicas and individual replicas.

== Groups of Flex Gateway Replicas

To view information about the status of groups of Flex Gateway replicas:

. From Anypoint Platform, select *Runtime Manager*.
. Select *Flex Gateway* in the navigation menu.

The *Name* column in the list of Flex Gateways lists the name and the number of connected replicas.

The *Status* column shows the connection status for the replicas associated with a particular gateway.

The following table describes the meaning of each status for a group of replicas:

[%header,cols="a,a"]
|===
|Status | Group of Replicas
|Connected | At least one replicas is to the platform and are serving traffic.
|Disconnected | There are no replicas connected to the platform and no information on whether they are serving traffic.
|===

== Individual Replicas

To view information about the status of groups of individual replicas:

To view information about the status of groups of Flex Gateway replicas:

. From Anypoint Platform, select *Runtime Manager*.
. Select *Flex Gateway* in the navigation menu.
. Select the name of the Flex Gateway whose individual replica statuses you want to view.

The *Status* column in the list of replicas shows the connection status for that particular replica.

The following table describes the meaning of each status for a single replica or group of replicas:

[%header,cols="20a,30a,30a"]
|===
|Status |Single Replica | Group of Replicas
|Connected | The replica is connected to the platform and is serving traffic. | All replicas are connected to the platform and are serving traffic.
|Disconnected |The replica is not connected to the platform. There is no information on whether it is serving traffic. Flex Gateway unregisters disconnected replicas after a period of inactivity. |There are no replicas connected to the platform and no information on whether they are serving traffic.
|Connected (x of y) | - |At least one (x) replica in this group is connected, while all other (y - x) replicas are disconnected.
|===
