The Policy Development Kit (PDK) provides example policy projects to get started configuring your Rust source code.

Find all policy examples and their documentation in the https://github.com/mulesoft/pdk-custom-policy-examples[PDK Custom Policy GitHub Repository^].

PDK offers the following example policies:

* https://github.com/mulesoft/pdk-custom-policy-examples/blob/main/query/README.md[Query Policy Example^]: Use the Query Policy as an example of how to handle DataWeave expressions in your custom policy.
* https://github.com/mulesoft/pdk-custom-policy-examples/blob/main/crypto/README.md[Crypto Policy Example^]: Use the Crypto Policy an an example of how to implement 3rd party libraries that provide cryptography capabilities in your custom policy.
* https://github.com/mulesoft/pdk-custom-policy-examples/tree/main/data-caching/README.md[Data Caching Policy Example^]: Use the Data Caching Policy an an example of how to implement data caching in your custom policy.
* https://github.com/mulesoft/pdk-custom-policy-examples/blob/main/jwt-validation/README.md[JWT Validation Policy Example^]: Use the JWT Validation Policy an an example of how to implement functions of the JWT Library in your custom policy.
* https://github.com/mulesoft/pdk-custom-policy-examples/blob/main/simple-oauth-2-validation/README.md[Simple OAuth 2.0 Validation Policy Example^]: Use the Simple OAuth 2.0 Validation Policy and an example of how to implement HTTP calls in your custom policy.
